Attendance Tracker provides the infomration about the number of students present in the classroom at particular time.

Each and every classrooms have their own WiFi routers, considering this fact we can identify the number of authenticated users connected to the wifi of the classroom. Depending on the number of devices each person is having, the number of students present in the classroom can be predicted. This application collects the log data from the server and with some modification of data displays the information on HTML page using flask app.

Primarily, this application is useful for Module coordinators who can find the number of students present in his/her class on particular day. It will definitely help them to understand the interests of the students in their modules.
